Dolphin Installation Guide

You need a dumped Wii NAND to use WiiMart on Dolphin. You can dump your Wii's NAND following the guides below:

  • How to dump your Wii's NAND
  • How to dump your vWii's NAND

Requirements

  1. Wii/vWii NAND dump.
  2. Dolphin Emulator (version 5.0-3416 or newer is recommended).
  3. WiiMart WAD and patched IOS WAD for your dumped NAND.

Installation Guide

1. Installing NAND on Dolphin:

  • Open Dolphin and go to Tools → Manage NAND → Import BootMii NAND Backup...
  • Keep in mind that merging a new NAND will replace all existing channels and save states. Click Yes to continue.
  • Select your nand.bin backup file.
  • Wait for the import process to finish.

2. Installing the Patched IOS and WiiMart:

  • In Dolphin, go to Tools → Install WAD.
  • Select IOS56.wad and wait for the installation to complete.
  • Repeat the process for WiiMart.wad.

3. Launching WiiMart:

  • Return to Dolphin's main menu and go to Tools → Load Wii System Menu 4.3x.
  • Find and launch WiiMart from the Wii Menu.
